# Alzheimer Prediction And Data Analysis

This project was developed to predict Alzheimer's disease probabilistically using clinical and demographic data. Correlations in the dataset were examined and various machine learning models were tested.


## Project Objective

* To predict disease risk by analyzing key features (age, gender, genetic factors, etc.) in the dataset.
* To apply probability theory and statistical methods to a real health dataset.


## Analysis Process

* **Exploratory Data Analysis (EDA):** A statistical summary of the dataset was extracted and missing data was removed.
* **Visualization:** Relationships between variables were visualized using the `matplotlib` and `seaborn` libraries.
* **Modeling:** Prediction models were created using classification algorithms.


## Tools Used

* **Python:** Main programming language.
* **Pandas & NumPy:** Data manipulation.
* **Scikit-learn:** Machine learning algorithms. * **Jupyter Notebook:** Interactive analysis and reporting.
